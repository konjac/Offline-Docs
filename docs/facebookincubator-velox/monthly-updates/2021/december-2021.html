December 2021 Update¶

Documentation¶

  • Add Aggregation-related optimizations guide.

  • Add missing documentation for json_extract_scalar() function.

  • Add documentation for missing functions in Mathematical Functions and Comparison Functions.

  • Update How to add an aggregate function? guide to reflect changes to the function registry.

  • Update How to add a scalar function? guide to describe how to write functions with complex type inputs and results.

Core Library¶

  • Add support for partition key aliases to Hive Connector.

  • Add support for conversions between Velox string and Arrow string.

  • Add StreamingAggregation operator.

  • Fix a bug in HashJoin filter to enable TPC-H query 19.

Presto Functions¶

  • Add corr(), covar_samp(), covar_pop(), every() aggregate functions.

  • Add from_base(), array_position() functions.

  • Add support for ‘timestamp with timezone’ in to_unixtime() function.
