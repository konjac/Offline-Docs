October 2021 Update¶

Documentation¶

  • Add new article: Plan Nodes and Operators.

Core Library¶

  • Enhance simple functions to

    • Allow returning an array of strings.

    • Process constant inputs once.

    • Access query configs.

    • Allow for providing fast path for all-ASCII inputs.

    More details in How to add a scalar function? guide.

  • Add support for RIGHT OUTER joins.

  • Add DATE type.

  • Add FixedSizeArray vector.

  • Add support for reading DWRF files from AWS S3 API compatible file systems, e.g. Minio and AWS-S3.

  • Add support for reading Parquet files using DuckDB’s Parquet reader.

Presto Functions¶

  • Add all remaining bitwise_xxx functions.

  • Add greatest() and least() functions.

  • Add log2() and log10() functions.

  • Add date_trunc(), day_of_week(), day_of_year(), year(), month(), day(), minute(), hour(), second() functions.

  • Add regexp_extract_all() function.

  • Add reverse(varchar), split() functions.

  • Add LIKE operator.
