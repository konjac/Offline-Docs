November 2022 Update¶

Documentation¶

  • Document spilling.

  • Document configuration properties.

  • Document expression and aggregation fuzzers.

Core Library¶

  • Add support for semi join project. #3068

  • Add support for using aggregate functions in window operator.

  • Add support for unbounded preceding, unbounded following, and current row frame bounds in range mode to window operator.

  • Add support for ROWS frame mode to window operator.

  • Add support for reporting ‘live’ runtime statistics while Task is running.

Presto Functions¶

  • Add strrpos() and truncate() scalar functions.

  • Add ntile() window function.

  • Add support for BOOLEAN and INTERVAL DAY TO SECOND inputs to min(), max() and histogram() aggregate functions.

  • Add support for BOOLEAN, TIMESTAMP, DATE and INTERVAL DAY TO SECOND input types to arbitrary() aggregate function.

Spark Functions¶

  • Add bitwise_and, bitwise_or, xxhash64 functions.

Performance and Correctness¶

  • Add aggregation and join fuzzers.

  • Add functions with variadic signatures to expression fuzzer.

  • Add COALESCE and IF special forms to expression fuzzer.

  • Add support for Lazy Vectors to expression fuzzer.

  • Add support to expression fuzzer for saving repro data for crashes. #3074
